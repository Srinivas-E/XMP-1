This code implements an SPI slave interface on an XMOS startKIT board to
communicate with a Raspberry Pi.

The example code here also implements PWM to control up to 8 servos based
on values send via SPI from the Raspberry Pi.


It was used to construct a simple robot platform called XMP-1.


See here for more information:
http://www.element14.com/community/community/raspberry-pi/raspberrypi_projects/blog/2014/05/22/xmos-startkit-building-an-xmos-and-raspberry-pi-robot-xmp-1
